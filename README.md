# Interactive Dyad Synthesis & Linkages

Course Project, ME252 | Prof. Anupam Saxena, IIT Kanpur (Jan'26 – May'26)

## Overview
An interactive JavaScript tool for four-bar linkage mechanism design using 
Burmester dyad-synthesis theory. Traces precision points across a linkage's 
full motion range while enforcing Grashof-validity for physically realizable motion.

## Features
- Analytical and graphical dyad-synthesis for four-bar linkage design
- Grashof-validity checks to constrain outputs to physically valid mechanisms
- Interactive canvas rendering: point placement, zoom, angle input, animation controls
- Traces 4-point precision synthesis, with an extension attempting 5-point synthesis
- Animated mechanism configurations validated against closed-form kinematic theory
